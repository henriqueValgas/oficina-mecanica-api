## Contexto

- O sistema precisa cadastra, clientes, funcionarios e fornecedores, 
todos compartilhando informaçoes comuns

## Decisão

- Criar entidade base pessoa, pessoa fisica e pessoa juridica,
cliente pessoa fisica funcionario, cliente pessoa juridica e fornecedores

## Justificativa

- elimina dados duplicados
- centraliza regras comuns
- facilita evolução do dominio
